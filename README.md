# DreamCanvas AI

A modern, web-based AI image generator powered by the **Pollinations.ai** API. This application allows users to generate high-quality images using various models (Flux, Turbo, etc.) with custom aspect ratios, all within a clean, responsive interface.

## ‚ú® Features
* **Multiple Models:** Choose between Flux, Turbo, Z-Image, and Seedream.
* **Aspect Ratio Control:** Easily switch between Square (1:1), Landscape (16:9), Portrait (9:16), and more.
* **Modern UI:** A responsive "Glassmorphism" design that works on desktop and mobile.
* **Instant Generation:** No login or API keys required for the user.
